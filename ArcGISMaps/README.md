### README for ArcGIS Maps SDK for JavaScript Example

This is a pure Javascript example for integrating Terrier into the ArcGIS Maps SDK for JavaScript.

To run this one, you'll need to serve this directory with some of web server.  I'll typically just use one integrated into VSCode, but anything will work.

The important bits are in the [startArcGISMap](startArcGISMap.js) file.  That's how you kick things off, but you'll need to include some other files.

These are described in the [Terrier Distribution](../doc/terrier-distribution.md) document.
