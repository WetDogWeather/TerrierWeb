# Terrier + OpenLayers + Vite

This example was constructed from the OpenLayers + Vite example and then we added Terrier for Web.

To run this in the local directory:

    npm start

The project isn't configured to deploy, so just use it in development as an example.

Then deploy the contents of the `dist` directory to your server.  You can also run `npm run serve` to serve the results of the `dist` directory for preview.
